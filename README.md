# React Backpack

![alt text](https://media3.giphy.com/media/mE5AQ8dqoH4Z2/giphy.gif)

## Includes
- Server Side Rendering
- SCSS modules
- Helmet
- Hot Reloading

## Run
```
# Development
npm run dev
# Build Production
npm run prodBuild
# Run Production
npm run prod
# Run pm2
npm run pm2
```
